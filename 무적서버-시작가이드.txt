🛡️ 무적서버 시작 가이드
========================

❌ 현재 상태: 무적서버 미실행
✅ 해결 방법: 다음 중 하나 선택하여 실행

방법 1) Git Bash 무적서버 (추천):
./scripts/git-ultimate-server-enhanced.sh

방법 2) Cursor 내장 터미널에서:
npm run dev

방법 3) PowerShell에서:
npm run dev:powershell

방법 4) 배치 파일 실행:
start-dev.bat

방법 5) PM2로 백그라운드 실행:
npm run dev:pm2

📊 무적서버 상태 확인:
- 포트 확인: netstat -ano | findstr :4900
- 브라우저 테스트: python scripts/quick-browser-test.py
- 실시간 모니터링: ./scripts/git-server-monitor.sh

🌐 성공하면:
http://localhost:4900/admin 접속 가능

💡 무적서버 특징:
- ✅ Next.js 서버 자동 시작
- ✅ 외부 킬 방지 (taskkill /f /im node.exe 대응)
- ✅ 자동 재시작 (최대 10회)
- ✅ 브라우저 자동 열기
- ✅ 실시간 로깅 및 모니터링
- ✅ 포트 충돌 자동 해결

🛡️ 보호 기능:
- 다른 터미널에서 node.exe 킬 당해도 자동 복구
- PM2보다 강력한 외부 간섭 방지
- Git Bash 환경에서 안정적 실행

💡 참고:
- 터미널 세션 문제로 자동 실행 불가
- 수동으로 위 명령어 중 하나를 실행해주세요
- Git Bash 무적서버가 가장 안정적입니다
- PM2 설정 오류 수정 완료 (ecosystem.config.js)
