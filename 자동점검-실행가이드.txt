🤖 자동 브라우저 점검 실행 가이드
=====================================

📋 준비된 스크립트: scripts\auto-browser-test.bat

🚀 실행 방법 (아무거나 선택):

방법 1) 명령 프롬프트에서:
scripts\auto-browser-test.bat

방법 2) PowerShell에서:
powershell -ExecutionPolicy Bypass -File scripts\run-auto-test.ps1

방법 3) 파일 탐색기에서:
scripts 폴더 → auto-browser-test.bat 더블클릭

📊 실행 과정:
[1단계] 서버 상태 사전 확인
[2단계] Chrome 브라우저 자동 열기 (admin 페이지)
[3단계] 자동 점검 실행 (연결/API/응답시간)
[4단계] 점검 결과 분석
[5단계] 테스트 브라우저 자동 종료
[6단계] 최종 보고서 출력

✅ 예상 결과:
- 브라우저가 자동으로 열림
- 몇 초간 점검 진행
- 결과 표시 후 브라우저 자동 종료
- 최종 보고서 출력

💡 참고:
- 무적서버가 실행 중이어야 합니다
- 점검 완료까지 약 15-20초 소요
- Chrome 브라우저가 설치되어 있어야 합니다
